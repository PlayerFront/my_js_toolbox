## Current structure
- src/
- |-- arrays (work with arrays)
 - --arrays-utils.js (simple array functions)
- |-- dom (DOM-related helpers)

- |-- Objects #work with objects
 - --objects-utils.js #basic object utilities

 - --sum.js (math operations)
